Zadanie 1

Napisz program w React, który posiada obsługę formularza do
wgrania pliku tekstowego, a następnie przestawi losowo szyk
liter w każdym wierszu oprócz pierwszej i ostatniej litery
wyrazu. Uwzględnij interpunkcję, wielkie/małe litery,
wielolinijkowe teksty, polskie znaki.


Zadanie 2

Napisz program w  React do walidacji numeru PESEL zgodnie z
oficjalną specyfikacją formatu. Przygotuj testy jednostkowe
sprawdzające kilka danych nieprawidłowych i przynajmniej jeden
poprawny numer PESEL.


Zadanie 3

Aplikacja powinna posiadać widok listy użytkowników (pobranych z

wpisów.
